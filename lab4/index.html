Lab 4 - Pseudocoding and Problem-Solving

Task 1: Pseudocode an everyday Task

With your partner, pseudocode an everyday task most students may not be familiar with. Examples include, changing a tire, making salsa, baking bread, getting an A in a class, or making art.

  1. Create a list of high-level tasks
  2. If necessary, break complicated tasks down into sub-steps.
  3. Put this in outline form.

Listed Pseudocode for Connect 4

  1. Print a 6x7 slot board
  2. Accept user input for each column
  3. Validate user move (if they are able to commit the move
  4. Move player's input pieve to lowest empty space in the column
  5. Evaluate possible moves for computer
  6. Choose computer's move
  7. Move computer's piece to lowest empty space in the column
  8. Confirm the winner when 4 pieces make a row, column, or diagonal
